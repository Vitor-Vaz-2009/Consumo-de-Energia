# ⚡ Calculadora de Consumo de Energia

## 📌 Sobre o projeto
Este projeto é uma calculadora em Python que estima o consumo mensal de energia elétrica de um aparelho com base no seu uso diário.

## 🛠️ Linguagem utilizada
- Python 🐍

## 📐 Fórmula utilizada

consumo = (potência × horas por dia × 30) / 1000

## ▶️ Como executar

1. Instale o Python
2. Execute o arquivo:

```bash
python app.py
